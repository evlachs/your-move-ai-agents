# Лаборатория цифровых агентов «ИИ-Синтез»

Образовательный проект для студентов — учимся проектировать ИИ-агентов и автоматизировать реальные задачи с помощью отечественных инструментов.

Проект реализуется в рамках Всероссийского студенческого конкурса **«Твой Ход»**, трек «Делаю», февраль — август 2026.

---

## Что здесь есть

Репозиторий — это публичная библиотека ИИ-агентов, созданных участниками лаборатории. Каждый агент живёт в отдельной папке и решает конкретную задачу.

```
/agents
  /github-monitor-agent      # Мониторинг репозитория: PR, коммиты → email-дайджест
  /smm-agent                 # Генерация постов для соц сетей
  /cv-review-agent           # Сравнение резюме с вакансией → структурированный отчёт
  ...
```

Каждый агент — не просто LLM-запрос. У каждого есть хотя бы один из трёх признаков настоящего агента: **инструменты** (внешние API, файлы, БД), **память** (состояние между запусками), **цикл** (observe → plan → act).

---

## Стек

Все агенты написаны на Python и используют отечественные ИИ-сервисы:

- **YandexGPT API** — языковые модели
- **Yandex Cloud** — деплой, инфраструктура
- **Yandex SpeechKit** — там где нужна речь

---

## Как устроена программа

Лаборатория работает модульно — от основ к собственному агенту:

1. Как устроены языковые модели и при чём тут агенты
2. Работа с API: YandexGPT, GitHub, внешние сервисы
3. Проектирование агентов: инструменты, память, циклы
4. Разработка собственного агента и защита на конкурсе
